CMDAR

Purpose

CMDAR is a simple tool that allows you to make your own command prompt without having to start from scratch.

Installation

There are steps on how to start playing around with your command prompt.

What you will need

You will need:

  • Terminal
  • Git

How to install

You can install by using Git and your terminal.

  1. Type this into your terminal: git clone https://github.com/akishore15/cmdar.git

There you go! It is installed. Now run it!

Run CMDAR

To run CMDAR, here are the materials you will need:

What you will need

You will need:

  • Terminal
  • Node.js

How to run

There are steps to run the CMDAR packet.

Steps to run:

  1. Open your terminal.
  2. Type in cd cmdar
  3. Type in node cli.js

You have officially booted up your own terminal!
